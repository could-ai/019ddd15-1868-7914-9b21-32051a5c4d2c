# ระบบลงทะเบียนบุคลากร (Personnel Registration Form)

แอปพลิเคชันลงทะเบียนบุคลากรที่พัฒนาด้วย Flutter ออกแบบมาในรูปแบบที่คล้ายกับ Google Form เพื่อให้บุคลากรสามารถเข้ามากรอกข้อมูลได้อย่างสะดวกและรวดเร็ว รองรับการทำงานแบบ Cross-platform โดยเฉพาะบน Desktop (Windows .exe)

## ฟีเจอร์หลัก (Features)

- **รหัสบุคลากร (Employee ID):** ช่องสำหรับกรอกรหัสพนักงาน
- **ชื่อ-สกุล (Name):** ระบบค้นหาอัตโนมัติ (Autocomplete) ช่วยให้สามารถเลือกชื่อที่มีอยู่ในระบบได้ทันที หรือพิมพ์ชื่อเองได้ในกรณีที่มีคำสั่งระบุเฉพาะหน่วยงานแต่ไม่มีชื่อในระบบ
- **หน่วยงานที่สังกัด (Department):** เลือกหน่วยงานจาก Dropdown หากไม่พบหน่วยงานของตนเอง สามารถเลือก "อื่นๆ" เพื่อกรอกชื่อหน่วยงานได้เอง
- **UI/UX ที่ใช้งานง่าย:** หน้าต่างฟอร์มที่สะอาดตา จัดวางให้อยู่กึ่งกลางหน้าจอเพื่อให้เหมาะสมกับการแสดงผลบน Desktop

## เครื่องมือที่ใช้ (Tech Stack)

- Framework: Flutter
- Language: Dart
- UI: Material Design 3

## วิธีการรันแอปพลิเคชัน (How to Run)

ตรวจสอบให้แน่ใจว่าได้ติดตั้ง Flutter SDK บนเครื่องของคุณแล้ว

1. ดึง dependency ทั้งหมดที่จำเป็น:
   ```bash
   flutter pub get
   ```
2. รันแอปพลิเคชัน:
   ```bash
   flutter run
   ```

## วิธีการสร้างไฟล์ Executable (.exe) สำหรับ Windows

แอปพลิเคชันนี้สามารถทำงานบน Windows เป็นไฟล์ `.exe` ได้อย่างสมบูรณ์แบบ ทำตามขั้นตอนด้านล่างนี้เพื่อสร้างไฟล์ `.exe`:

1. ตรวจสอบให้แน่ใจว่าเครื่องคอมพิวเตอร์ของคุณมี Visual Studio พร้อม C++ workload ติดตั้งอยู่ (จำเป็นสำหรับการบิวด์แอปพลิเคชัน Flutter บน Windows)
2. เปิดการรองรับ Windows ให้กับโปรเจ็กต์:
   ```bash
   flutter config --enable-windows-desktop
   ```
3. บิวด์ไฟล์ Windows:
   ```bash
   flutter build windows
   ```
4. ไฟล์ Executable (.exe) ที่สร้างเสร็จแล้วจะอยู่ที่:
   `build/windows/x64/runner/Release/`
   
   คุณสามารถคัดลอกไฟล์ในโฟลเดอร์ Release ทั้งหมดไปให้ผู้ใช้อื่นเพื่อเริ่มใช้งานแอปพลิเคชันได้ทันที
